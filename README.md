This project is a robust platform for hospital information management and image comunication systems (RIS/PACS). It implements _Clean Architecture_ principles to ensure that businss logic is independent of frameworks, user interface, and dataBase.

## Tech Stack

- **Backend:** Python 3.13>=, FastAPI, SQLAlchemy.
- **Frontend:** React.
- **Architecture:** Clean Architecture.
- **DataBase:** PostgreSQL.

## Features

- Patient and medical appointment management.
- Module RIS (**Radiology Information System**) for managing radiology data and workflows.
- Conceptual integration of PACS (**Picture Archiving and Communication System**) for handling medical images.
- Authentication and authorization system based on roles _(**RBAC**)_.
